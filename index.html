mRMR Feature Selection

This project aims to predict potential threats by applying feature selection techniques, specifically Maximum Relevancy Minimum Redundancy (mRMR), on a dataset. The process begins by scaling the data, followed by calculating the mutual information and correlation scores to assess feature relevancy and redundancy. The selected features are then used to build a model for threat prediction. This approach ensures that the most informative and least redundant features are utilized, enhancing the prediction accuracy. This said, the code in the project can also be used for other prediction projects which use the mRMR algorith of feature selection.
